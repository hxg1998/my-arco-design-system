# Acceptance Checklist

- The page uses Arco primitives wherever practical.
- Spacing and typography match the defined token system.
- Empty, loading, disabled, and error states are covered.
- Forms provide validation and clear submit feedback.
- Tables support sorting, filtering, or bulk actions where needed.
- The layout remains usable on common laptop widths.
- Accessibility basics are checked for focus, contrast, and labeling.
